# Foxsir-BDSM v1.0 启动器

本版本为根基版本，仅包含 Control Center（启动器）、身份认证、权限分级和项目导航中心，所有业务子项目均为空占位。

## 快速开始

1. 修改 `assets/js/config.js` 中的 Supabase URL 和 Anon Key。
2. 使用 Live Server 或任意静态服务器打开 `index.html`。
3. 在 Supabase Dashboard 中为用户设置 `raw_user_meta_data` 的 `role` 字段（self / subadmin / admin）以测试不同权限。
